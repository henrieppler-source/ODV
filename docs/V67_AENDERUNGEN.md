# V67 Änderungen

## Stammdatenblatt für Personenbilder

Für Bilddateien mit Personenzuordnung kann per Rechtsklick ein A4-Stammdatenblatt als PDF erzeugt werden. Das Blatt enthält oben das Bild mit sichtbaren Personenpunkten und unten eine Personenliste sowie zentrale Dateidaten.

## Download/Kopie speichern

Dateien können per Rechtsklick als Kopie in den lokalen Downloads-Ordner oder ein anderes gewähltes Verzeichnis gespeichert werden. Namenskonflikte werden über Zähler vermieden.

## Dokumentzugriffe

Öffnen und Download/Kopie von ODV-Dokumenten werden serverseitig in `document_history` protokolliert. Unter Auswertungen gibt es die neue Übersicht „Dokumentzugriffe...“.

Hinweis: Protokolliert werden nur Aktionen über die ODV-App, nicht direkte Zugriffe über Windows Explorer oder Nextcloud.

## Layoutkorrektur

Die Überlagerung im Metadatenbereich von „Dateien anzeigen“ wurde beseitigt; der Hinweis steht nun getrennt über dem Formular.
