# V46 – kleine Bedienkorrekturen

- Dashboard aktualisiert sofort beim Umschalten zwischen Alle Aktionen und Nur eigene Aktionen.
- Beim Wechsel auf Dateien anzeigen wird keine alte Datei/Vorschau mehr angezeigt; Auswahl erst nach Klick.
- Beim Wechsel auf Dateien bearbeiten wird keine alte Datei/Vorschau mehr angezeigt; Auswahl erst nach Klick.
- Standardrechte enthalten 03_INFORMATION für Ortschronisten als Leserecht.

Hinweis Admin-Menü:
Die Admin-Menüpunkte sind bewusst manuelle Werkzeuge: Admin-Einstellungen, Ortsordner-Stammdaten, vorhandene Dateien einlesen und lokale Sicherungen bereinigen werden nicht automatisch ausgeführt.
