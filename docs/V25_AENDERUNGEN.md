# V25 Änderungen

- Zielordner-Auswahl kann jetzt zusätzlich über einen Baumdialog erfolgen.
- Upload: Button `Baum...` neben `Zielordner Nextcloud`.
- Admin-Bearbeitung: Button `Baum...` neben `Datei ablegen in`.
- Der Baum zeigt die Ordnerhierarchie übersichtlich an.
- Graue Einträge dienen nur der Struktur; auswählbar sind nur beschreibbare Zielordner.
- Doppelklick auf einen auswählbaren Ordner übernimmt den Ordner.
- Zielordnerlisten werden alphabetisch sortiert.

Hinweis: Die bisherige Auswahlliste bleibt bestehen, damit häufig genutzte Zielordner weiterhin schnell ausgewählt werden können.
