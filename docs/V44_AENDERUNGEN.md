# V44 Änderungen / Komfort- und Admin-Fixes

- Mausrad-Scrollen in Listen, Textfeldern und Baumansichten verbessert.
- Lokale Pfade werden im Betriebssystemformat vereinheitlicht.
- Admin-Einstellungen werden mit Standardwerten vorbelegt.
- Vorhandene Dateien einlesen scannt automatisch; der Button „Prüfen“ entfällt.
- Benutzerverwaltung: Passwort kann per gedrücktem Auge-Button kurz sichtbar gemacht werden.
- Dateien hochladen: Ordnerupload ergänzt; Metadaten werden einmal erfasst und für alle Dateien verwendet.
- Dateien anzeigen: Startverzeichnis zeigt das Nextcloud-Stammverzeichnis, Aktualisierung beim Reiterwechsel, Baumauswahl ergänzt.
- Dateien bearbeiten: lokale Pfade anderer Benutzer werden auf das eigene Nextcloud-Stammverzeichnis umgerechnet, damit Admins fremde Uploads öffnen/vorschauen/bearbeiten können.
- Vorschau für JPG/PNG in Dateien bearbeiten robuster.
- Dokumenttyp wird aus der Dateiendung nachgezogen, wenn er fehlt.
- Normierter Admin-Dateiname jetzt: Datum_Ort_Dateiname.
- Mehrere PDF-Dateien können in der Adminliste zusammengeführt werden.
