# Buildfix 4 – Komfort-Updater

Der Komfort-Updater wird jetzt erst nach Bestätigung des Hinweisfensters gestartet.
Damit beginnt das Ersetzen des Programmordners nicht mehr, solange die alte ODV wegen des Meldungsfensters noch aktiv ist.

Ablauf: Update vorbereiten → Hinweis anzeigen → OK → Updater starten → ODV beendet sich hart → Updater ersetzt Dateien → neue ODV startet.
