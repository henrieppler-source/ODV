# Server v41

Für v41 ist keine serverseitige Änderung gegenüber v40 erforderlich.

Die Desktop-App erzeugt bei Versandart „Nextcloud-Downloadlink versenden“ für mehrere ausgewählte Dokumente jeweils einen Downloadlink und schreibt diese Linkliste direkt in den Mailtext.

Falls auf dem Server bereits `routes_v40.php` als `ortschronik-api/routes.php` aktiv ist, muss für v41 nichts am Server geändert werden.
